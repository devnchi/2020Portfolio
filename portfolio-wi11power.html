Wi11power App

Home /To-Do Today!

Project Details

This is the beginning of a productivity app that I am making with React, Express and MySQL. I call it the "Wi11power App" because stylizing words with the number 11 is something that my brand has been doing as a distingushing quality. Whenever I start a developement, I think of it as a product, and all the nuances that go into forming a cohesive and attractive one.

This list application, in my opinion, is a good start to a more dynamic but simple list-making/task-tracking web application. Additionally it's very cool to see how knowing how to build and program technology can make life's more minute day-to-day tasks and activities become more dynamic and engaging. Looking at this To-Do List makes me want to use it.

The functionality of this development includes the ability to add list items, edit list items, cross them out, and track how many are left. This development was also made with a responsive layout so that it looks good on laptops, tablets, and mobile devices. Give it a go at the link below!

  • Skills Used: MySQL, React.js, Express,js, Design, Front-end, Javascript, JSX, Photoshop, Illustrator
